Importar Base de Leads (Llamadas)

Sube el archivo de Llamadas/Agendas. El sistema usará el Email como identificador único para crear usuarios y perfiles.

Arrastra tu archivo CSV aquí

o haz click para seleccionar

Reglas de Importación

  • Si el Email no existe, se crea un nuevo Usuario.
  • Si ya existe, se actualizan sus datos de contacto.
  • Se crea un registro de Cita para cada fila.
  • Los duplicados de email se unifican bajo el mismo usuario.

Importar Ventas y Vincular

Sube el archivo de Ventas. El sistema buscará el Email en la base de datos de Leads creada en el paso 1.

Arrastra tu archivo CSV aquí

o haz click para seleccionar

Reglas de Vinculación

  • Se busca al usuario por Email.
  • Si se encuentra, se registra la venta a su nombre.
  • Si NO se encuentra: Se crea una "Venta Huérfana".
  • Podrás asignar manualmente las ventas huérfanas después.
Gestionar Ventas Huérfanas

¿Estás absolutamente seguro?

Esta acción eliminará TODOS los leads, ventas, citas, pagos y reportes de la base de datos. Solo se conservarán las cuentas de Administrador y Closers.

Esta acción no se puede deshacer.
